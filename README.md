# 🌿 Wellness Mindset Coach Chatbot

A simple command-line wellness coach inspired by Marisa Peer with the core message **"I am enough"**.

This project is a **custom pathway** to OpenAI — it guides and shapes the AI to act as a supportive mindset coach.

## Setup

1. Clone the repo
2. Create virtual environment:
   
   python -m venv venv
   venv\Scripts\activate     # Windows

Install dependencies

pip install -r requirements.txt

Set up API key:Copy .env.example → .env (or chatbot.env)
Paste your OpenAI API key inside

Run it:

python wellness_chatbot.py

Notes

You need an OpenAI account with credits added (add $5–10 at platform.openai.com)
The coach remembers the conversation during one session
Made as a personal project for self-growth and learning.
